Alarm CLI
=========================

Alarm CLI can be used to interact with alarm service using a command line interface.

Using alarm cli, you can perform following actions:
* init 
* list 
* acknowledge 
* unacknowledge 
* activate 
* deactivate 
* shelve 
* unshelve 
* reset 
* severity 
* health

If you want to get started with using alarm CLI, refer these [examples](https://tmtsoftware.github.io/csw/apps/cswalarmcli.html).
